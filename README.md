## ホームメインビジュアルの設定手順
1. WordPress 管理画面にログインします。
2. 左メニューの **「ホーム設定」**（ACF オプションページ）を開きます。
3. 「メインビジュアルスライド」リピーターで「スライドを追加」を押し、以下を設定します。
   - **スライドタイプ**: 投稿・固定ページ・カスタム投稿を使う場合は「投稿 / 固定ページ / カスタム投稿」を選択し、「関連投稿」で対象のコンテンツを指定します。
   - **バナーリンク**: 任意の URL を使いたい場合は「バナーリンク」を選択し、「リンク先URL」に遷移先を入力します。
4. 必要に応じてサブタイトル・タイトル上書き・メイン画像・ボタンテキストを入力します（未入力時は投稿情報やアイキャッチを自動で利用します）。
5. スライドを追加・並べ替えたら保存すると、トップページのメインビジュアルカルーセルに反映されます。
